# 香港長者醫療券配偶共用計劃：續用離世配偶醫療券涉詐騙案深度分析

**資料來源**：am730 / 醫衛局書面回覆立法會議員提問  
**報道日期**：2026年5月6日  
**分析日期**：2026年5月  

---

## 1. 核心問題與動機

### 政策背景
- 香港政府推出**長者醫療券**（Elderly Health Care Voucher），資助65歲或以上長者使用私營醫療服務。
- **2023年7月起**擴大實施**配偶共用機制**：65歲或以上長者夫婦可登記連結戶口，一方醫療券餘額用盡後可使用配偶戶口。
- 使用規則：每次須出示配偶身份證副本核對。
- 截至2025年12月底數據：
  - 逾 **17.8萬對** 長者配偶完成連結（合共35.6萬人）
  - 涉及約 **67萬宗交易**，總申領金額約 **4.1億元**

### 核心問題
- 長者**離世後醫療券即時失效**，餘額**不可轉移、繼承或繼續使用**。
- 任何人（包括家人）繼續使用已故配偶的醫療券（出示已故者身份證），均屬違法，可能構成**欺詐罪**。
- 衛生署監察發現 **383宗懷疑不當使用個案**，涉及 **524宗交易**（佔總交易約 **0.08%**）。
  - **351宗**（91.6%）已轉介警方跟進。
  - 其餘32宗仍在調查中。
- 同期已有 **1宗成功檢控**：被告欺詐罪成，判處罰款及緩刑監禁。

### 主要動機分析
- **經濟因素**：節省私營醫療費用（診所、牙科、物理治療等）。
- **認知不足**：部分長者或家人誤以為配偶共用可「繼承」或不知離世後失效。
- **操作漏洞**：服務提供者核對身份證不嚴謹；死亡通報機制早期未完善。
- **家庭壓力**：長者離世後，家人面對醫療開支壓力，出現「不用白不用」心態。
- **系統性挑戰**：高齡化社會下，福利便利性與防濫用之間的平衡難題。

## 2. 結果與政府應對措施

### 量化統計

| 指標               | 數據          | 比例 / 說明                  |
|--------------------|---------------|------------------------------|
| 懷疑不當使用個案   | 383宗        | 涉及524宗交易                |
| 轉介警方           | 351宗        | 佔91.6%                      |
| 成功檢控           | 1宗          | 罰款 + 緩刑監禁              |
| 總交易宗數         | 67萬宗       | 涉案佔0.08%                  |
| 已連結配偶戶口     | 17.8萬對     | 合共35.6萬人                 |
| 總申領金額         | 約4.1億元    | -                            |

### 政府改善措施
- 與入境處、醫管局建立死亡通報系統。
- **2024年6月起**：實施**每小時離世資料通報機制**，醫健通已登記長者在公院離世即時凍結醫療券。
- 持續主動監察 + 處理舉報。
- 針對服務提供者：截至2026年4月底處理31宗違規個案（警告、停付款、追討款項）。
- 整體顯示計劃運行良好，但需持續優化。

## 3. 分析與洞見

### 正面意義
- 配偶共用計劃大幅提升長者醫療可及性，惠及35.6萬人，體現「以人為本」福利政策。
- 涉案比例極低（0.08%），證明公帑整體使用妥善。
- 每小時死亡通報機制是重要的技術進步，有效縮短濫用窗口期。

### 風險與漏洞
- **人性灰色地帶**：多數個案可能屬「不知情」或情感驅使，而非蓄意詐騙，但法律上仍構成犯罪。
- **執行困難**：
  - 私營診所身份核對責任。
  - 子女/親屬代用情況。
  - 死亡通報覆蓋率需達100%。
- **社會影響**：
  - 即使比例低，仍可能損害福利計劃公信力。
  - 對離世家庭造成額外心理與經濟壓力。
  - 反映香港高齡化社會中福利政策的共通挑戰。

### 多角度洞見
- **便利 vs 監管**：共用機制提升使用率，但同時增加監管複雜度。
- **科技 vs 人性**：自動凍結有效，但無法完全取代宣傳教育。
- **邊緣情況**：突然離世、認知障礙家庭、跨境個案、服務提供者共謀等需特別處理。
- **政策啟示**：好政策需兼顧執行細節與人文關懷，否則易衍生爭議。

---

## 4. 結論與建議

**結論**：  
長者醫療券配偶共用計劃整體成功，覆蓋廣泛、惠及眾多，但離世後續用問題暴露了宣傳、執行與監管上的不足。383宗個案（351宗已交警方）雖然比例低，卻凸顯在高齡化社會中，福利政策必須同時兼顧「便利性」與「可持續防濫用」。政府已積極優化死亡通報機制，顯示正面態度，但單靠技術與執法並不足夠，需配合教育及人文支援。

**政策建議**：
1. **加強宣傳教育**：離世前後自動SMS/醫健通通知；社區中心、診所派發清晰指引；新登記戶口強制教育。
2. **技術優化**：擴大醫健通即時凍結範圍；探索生物識別或數碼身份驗證。
3. **人文支援**：為離世家庭提供短暫過渡期或輔導；簡化剩餘金額處理流程。
4. **透明公開**：定期發布統計數據，維持公眾信任。
5. **跨部門合作**：醫衛局、衛生署、警方、社署聯動，對非蓄意個案考慮從輕處理。

---

**整體啟示**：  
香港作為全球高齡化前沿城市，醫療福利政策的成敗取決於「精準監管」與「社會共識」的平衡。此案例是典型「好政策遇執行挑戰」的示範，值得持續跟進與優化。
